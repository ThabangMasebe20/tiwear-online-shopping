# 🛍️ TiWear – Online Shopping Website

TiWear is a responsive front-end e-commerce website developed to simulate a modern online shopping experience. The platform showcases a variety of tech accessories, wearables, and lifestyle products with an interactive UI and dynamic cart functionality.

This project is actively maintained and continuously improved with new features, UI enhancements, and bug fixes.

---

## 🌐 Overview

TiWear provides users with a clean and engaging shopping interface, allowing them to:

* Browse featured and latest products
* View product details
* Add items to a shopping cart
* Navigate across multiple pages
* Subscribe to newsletters

---

## ⚙️ How It Works

* Products are defined using HTML with `data-name`, `data-price`, and `data-img`
* JavaScript handles:

  * Cart updates
  * Notifications (toast messages)
  * Mobile navigation toggle
* UI updates dynamically without backend integration
